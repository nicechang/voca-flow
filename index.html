데이터 관리

💡 구글 시트 연동 설치 가이드

  1. 새 구글 스프레드시트를 생성합니다.
  2. 상단 메뉴에서 확장 프로그램 (Extensions) > Apps Script를 클릭합니다.
  3. 프로젝트에 생성된 코드 에디터의 기존 코드를 지우고, 이 웹앱 작업 디렉토리의 google_script.js 소스 코드를 복사해서 붙여넣습니다.
  4. 상단 저장 (Save) 버튼을 클릭합니다.
  5. 우측 상단 배포 (Deploy) > 새 배포 (New deployment)를 선택합니다.
  6. 유형 유형에서 웹 앱 (Web app)을 선택합니다.
  7. 설정값:
    • 다음 사용자 권한으로 실행: 나 (Me)
    • 액세스할 수 있는 사용자: 모든 사용자 (Anyone) (로그인 없이 스마트폰에서 연동하기 위해 필수)
  8. 배포 (Deploy) 버튼을 누르고 권한 검토를 진행하여 허용해 줍니다.
  9. 화면에 출력되는 웹 앱 URL 주소를 복사해 위의 입력란에 붙여넣습니다.
