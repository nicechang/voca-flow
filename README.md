# VocaFlow - 개인 영어 암기 앱 (English Memorizer)

VocaFlow는 네이버 영한사전 API를 연동하여 단어의 뜻, 유사어, 예문/표현을 편리하게 수집하고, 망각곡선 정답률 가중치를 기반으로 모바일 환경에서 학습할 수 있는 프리미엄 단어장 앱입니다.

구글 스프레드시트(Google Apps Script)와 연동되어 PC와 스마트폰 등 기기에 구애받지 않고 언제 어디서나 데이터를 실시간으로 동기화합니다.

---

## ✨ 주요 기능 (Key Features)

1. **간편한 단어 검색 및 자동 파싱**:
   * 단어/숙어를 검색하면 네이버 사전에서 발음 기호, 주요 한글 의미, 유사어(Synonyms), 많이 활용되는 표현(Collocations)을 실시간으로 가져옵니다.
   * 복잡한 타이핑 대신, 추천 단어 뜻의 체크박스를 터치하여 바로 나의 의미장으로 추가할 수 있습니다.
2. **모바일에 최적화된 3가지 학습 모드**:
   * **플래시 카드**: 3D 카드 뒤집기 애니메이션과 함께 원어민 발음(TTS)을 청취하며 '알겠음 / 모르겠음'으로 학습을 진행합니다.
   * **뜻 고르기(객관식)**: 단어장의 오답들을 보기로 한 4지선다형 뜻 고르기 퀴즈입니다.
   * **예문 빈칸 채우기**: 네이버 사전의 예문 속 단어를 빈칸(`[____]`)으로 변경하여 뉘앙스를 학습합니다.
3. **스마트 우선순위 가중치 학습**:
   * 단어의 누적 오답 횟수를 실시간 분석하여 취약한 단어들이 카드 학습 및 퀴즈 세션에 높은 확률로 먼저 등장하도록 유도합니다.
4. **구글 스프레드시트 양방향 동기화**:
   * Google Apps Script 웹앱 엔드포인트를 매개로, 로컬 브라우저 데이터와 구글 스프레드시트 데이터를 타임스탬프 기반으로 안전하게 병합합니다.
5. **독립 로컬 서버 실행**:
   * 오프라인 또는 인터넷 연결이 원활하지 않은 로컬 PC 환경에서도 디스크 파일 DB(`database.json`)에 데이터를 저장할 수 있습니다.

## 🚀 시작 가이드 (Quick Start)

### 1. 로컬 환경에서 실행
PC에서 로컬 서버를 구동하여 데이터 관리 및 Naver API 프록시를 켭니다.
```bash
python3 server.py
```
서버 실행 후 브라우저에서 **`http://localhost:8000`**으로 접속합니다.

### 2. 구글 스프레드시트 동기화 설정
1. 새 Google Spreadsheet를 생성합니다.
2. **확장 프로그램** ➔ **Apps Script**를 누르고 에디터의 기존 코드를 지운 후, 이 프로젝트의 `google_script.js` 파일 내용을 전체 복사하여 붙여넣고 저장합니다.
3. 오른쪽 상단 **배포** ➔ **새 배포** 클릭:
   * **유형**: 웹 앱 (Web app)
   * **다음 사용자 권한으로 실행**: 나 (Me)
   * **액세스할 수 있는 사용자**: 모든 사용자 (Anyone)
4. 배포가 완료되면 **웹 앱 URL** 주소를 복사합니다.
5. VocaFlow 웹사이트의 **설정(Settings)** ➔ 데이터 관리 방식을 **구글 시트 연동**으로 선택한 뒤 복사한 웹 앱 URL을 붙여넣고 **양방향 동기화 실행**을 누릅니다.

### 3. 모바일 앱(PWA) 추가
1. GitHub Pages 배포 URL로 모바일 기기 브라우저에서 접속합니다.
2. 스마트폰 브라우저 메뉴에서 **홈 화면에 추가(Add to Home Screen)**를 선택합니다.
3. 바탕화면에 설치된 아이콘을 누르면 브라우저 주소창 없이 네이티브 앱 형태로 켜집니다.
